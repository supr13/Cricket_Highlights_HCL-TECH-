# Cricket_Highlights_HCL-TECH-

Automatically generates highlights from full-length cricket match videos
Focuses on key events like fours, sixes, and wickets
Uses a simple CSV-based event input for easy customization
Extracts short clips around important match moments
Adds clear text labels for better understanding
Merges all clips into a single engaging highlights video
Reduces long match duration into quick watchable content
Demonstrates video processing and event-based filtering
Designed to be simple, efficient, and beginner-friendly
Suitable for real-world applications in sports media and content creation
